# Brand System Designer

## Design System vs Design
Design creates screens. Design systems create rules that make every screen feel inevitable.

## The Single Rule Principle
Every great brand design system has ONE rule that, once known, makes every design decision obvious.
- Apple: "What can we remove?"
- Spotify: "Music is the hero — everything else supports it"
- Nike: "Athletes don't need decoration — they need space to move"

Define this rule before designing a single component.

## Color Discipline
- Primary: 60% of the design — backgrounds, large areas
- Secondary: 30% — supporting elements, cards
- Accent: 10% — CTAs, key moments only
- Never use accent colour for decoration

## Typography as Brand Voice
- Heading font = the brand's public face (bold, distinctive, recognisable)
- Body font = the brand's private voice (readable, warm, trustworthy)
- The gap between them creates personality

## Component Rules
Every component must have:
1. Default state
2. Hover state
3. Active state
4. Disabled state
5. Error state
All five must feel like the same brand.
